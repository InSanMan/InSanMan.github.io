了解ChatGPT

ChatGPT

官网：https://chat.openai.com/

随着科技的发展，人工智能（Artificial Intelligence，简称AI）已经逐渐成为了我们生活中的常态

人工智能（AI）是一种模拟人类智能思维的技术，它可以实现人类的认知和思维活动。通过这种技术，计算机可以模拟人类的思维方式和智能，从而可以完成许多复杂的任务，如图像识别、语音识别、自然语言处理、决策制定等。AI技术包括许多不同的领域和技术，例如机器学习、深度学习、自然语言处理、计算机视觉等，这些技术都是为了实现人工智能而发展起来的。

AI的工作方式可以概括为三个部分：输入、处理和输出。

而ChatGPT 是 OpenAI 开发的人工智能聊天机器人。ChatGPT在对话能力表现比传统的对话系统(微软小冰、Alexa)更加惊艳,其惊艳的表现，得益 于大模型的“涌现”能力

你可以简单的理解为 AI对话软件，你可以向ChatGPT提问，告诉他你想让他做什么。但是，想让ChatGPT帮我们做更多的事情，就必须要学会让ChatGPT听我们的话，这就是提示词（Prompt）

Prompt

一、什么是Prompt?

Prompt的概念我们在使用GPT的时候,给GPT发送的消息就是Prompt.

​ 例如,当我们问ChatGPT”WPS是什么软件?”时:

其中,”WPS是什么软件?”这句话就是Prompt.

二、为rompt的4个原则

  1. Instruction： 指令，即我们想要GPT做的事情。
  2. Context： 背景信息，也可以说是上下文信息,告诉GPT现在是什么情况.这可以引导GPT做出更好的反应。
  3. Input Data： 输入数据，告知GPT需要处理的数据和内容。
  4. Output Indicator： 输出规则，告知GPT我它需要按什么样子回答。

我们来进行一场扮演，你要装成是丹尼尔的《科学学习》。要求: 你只是这本书,你不是ChatGPT，你没有除了这本书以外的任何知识,不要说任何无关的东西，不知道就说不知道。将你的答案分点给出,我的第一个问题是:”睡眠是如何影响学习的?”

​ 在这个案例中,Prompt的背景是”ChatGPT是书籍<科学学习>”,我们在这里交代了ChatGPT的身份.

​ 我们对ChatGPT下达的指令是:”要求: 你只是这本书,你不是ChatGPT，你没有除了这本书以外的任何知识,不要说任何无关的东西，不知道就说不知道。将你的答案分点给出.”

​ “将你的答案分点给出”这里指明了ChatGPT的输出格式,ChatGPT按四点给出了回答.

​ “我的第一个问题是:”睡眠时如何影响学习的?”是我们给ChatGPT提供的数据.

​

主流AI工具

2023年开年以来，AI似乎顿悟了一样，突然间变得非常智能，非常好用了，在这一节，我将会向大家介绍一下目前常见的AI工具。主要是比较热门的两部分，文本生成类的AI工具和AI绘画工具。

文本生成类AI工具

在文字生成能力上，ChatGPT是非常强的，很多公众号、小红书号主等都用它来写文章，润色内容，提高内容生产速度。一些企业白领用它来生成生成邮件内容、总结会议记录等。正因为它的能力广泛而且好用，即便是OpenAI的原因，不允许部分地区的用户使用，也挡不住大家使用ChatGPT的热情。

在这一节中，主要向大家介绍类似ChatGPT的工具，我们在这里介绍的是正规军(有公司背书，不是私人搭建的网站)。

ChatGPT爆火以来，各种小程序、H5页面接入了GPT-3.5的模型，然后给大家使用，通过限制次数的方式收费。

这类型大家使用的时候要千万注意！

你发给他们的数据通常他们在后台可以看得到，知道你发的是什么内容。

所以，对于这种第三方的工具，我们简单体验即可，千万不要发敏感信息!!!

好了，接下来我们就介绍各种类似ChatGPT的工具

ChatGPT

我们先介绍下当前最先进的大语言模型–ChatGPT，ChatGPT作为世界上第一个公布的，能称得上是“人工智能”的模型，刚出来就火了。我们来看看它的界面，由于OpenAI没有对国内开放，所以，国内的小伙伴使用起来真的是困难重重。没有任何技术基础的小伙伴不太建议强行上手，强行上手费钱，而且也有很大几率用不了多久就会被OpenAI封，具体还是看看自己的选择吧。ChatGPT的官网为 chat.openai.com，如果打不开的小伙伴，说明你所在的区域OpenAI不允许使用。

官方地址为: chat.openai.com

New Bing

New Bing底层用的模型和GPT-3.5是一样的，是微软开发的。说起来微软还是OpenAI的大股东，所以OpenAI的ChatGPT的惊艳表现展现给全世界后，微软就赶紧把GPT模型集成到自家的搜索上了。不过呢，New Bing更专注于搜索，能够联网。这点还是和ChatGPT不太一样的。

官方地址为: www.bing.com/new

上面是New Bing的界面，New Bing其实使用起来并不比ChatGPT简单多少，New Bing和OpenAI有一样的区域限制，所以也不是很建议新手小伙伴使用(有能力的小伙伴除外)

ZelinAI

ZelinAI是取得了微软授权，可以商业化调用GPT模型的应用，目前，已经有12万月活。ZelinAI想对于ChatGPT和New Bing来说，他们的底层调用的都是GPT模型，其能力都差不多。

最重要的一点是，ZelinAi能够在直接访问，没有区域限制，注册就能使用，所以对我们的新手同学上手是比较友好的。另外ZelinAI支持每个人设置调整模型，创建自己的垂直领域的应用，然后将应用分享给其他人，赚取收益。这无疑比ChatGPT和New Bing更有趣了。

相关使用请参考我们的《ZelinAI教程》

官方地址为:www.zelinai.com

Claude

Claude我整体感觉还不错，响应速度、生成内容也算丰富。Claude也是外国公司开发的，他也没有对国内开放，所以使用起来也有一定的不便之处。

Claude和New Bing、ChatGPT等不一样，他没有一个独立的网页，它是集成到Slack使用的。我们来看看他的界面，在这里，我让他参考关于ZelinAI的介绍写了一段文案。大家有没有发现，Claude太会吹牛逼了，竟然说ZelinAI有120万的月活用户! 所以大家使用的时候，生成内容一定要注意审核。

官方地址为: https://www.anthropic.com

至于如何安装，大家自行搜索即可，网上有太多类似的教程了。

Google Bard

Bard 是谷歌开发的，然而。。。不幸的是，目前他只支持英语、日语和韩语，所以大家了解即可，不做过多介绍了。

上面提及的都是国外的大语言模型，作为互联网大国，我们国家也是不赖的。接下来就说我们国内的吧

百度的文言一心

百度作为我国最早推出大语言模型平台的公司，在与ChatGPT相似的功能基础上，还额外增加了绘画功能。这使得文心一言不仅能为用户提供语言方面的支持，同时还能满足创意绘画的需求，为用户带来更丰富多样的体验。

图1-3 百度文言一心

官网:https://yiyan.baidu.com/

阿里的通义千问

作为中国顶尖的互联网企业，阿里巴巴在人工智能领域毫不逊色。2023年4月11日，阿里巴巴推出了大语言模型“通义千问”，展现了其在该领域的实力。阿里巴巴的CEO宣布，未来公司将全面整合“通义千问”大模型至旗下各项产品中，以实现产品的智能化升级，进一步提升用户体验。

官网 :https://tongyi.aliyun.com/

科大讯飞的星火

科大讯飞，作为中国AI科技的领军企业，于2023年4月20日宣布将在同年的5月6日发布其大语言模型产品。未来，该大模型将逐步推出针对不同行业的解决方案，拓展其应用领域，展现出广阔的发展前景。

毫无疑问，以上列举的仅是部分大语言模型，事实上还有许多其他公司推出的大模型未被提及。这些现象无不证明了AI时代的到来，预示着未来将带来更多的创新与变革。

官网: https://xinghuo.xfyun.cn

以上就是听的比较多的文本生成类AI工具了，当然，AI生成类工具并不仅仅限于以上的，还有更多的并没有列出来。

AI绘画类:

AI绘画可以说是今天爆火的其中一个AI赛道了。要知道，还在前些年，AI绘画画出来的东西非常搞笑，各种奇怪的画面，让人刷新了三观，所以当时别人觉得AI绘画就是个“人工智障”。然而才没多久，AI绘画的作品跟艺术大师有的一拼了，一些设计师甚至因为AI绘画的出现被迫离职! AI真的是太快了，前两年还觉得这种是异想天开的事儿，现在居然都实现了。

下面我们来介绍一下各种主流的AI绘画工具!!

Midjourney

midjourney是当前最火的AI绘画工具了，其画出来的作品非常具有艺术风格，据说midjourney的团队不超过20个人!

mijourney虽然好用，但是购买一个标准账号价格并不低，需要30$。关于如何使用、如何充值不在此作太多介绍，可到我们的AI绘画课程学习。

官网地址: https://www.midjourney.com/home/?callbackUrl=%2Fapp%2F

Stable diffusion

和Midjourney高昂价格相对应的是，Stable Diffusion是免费的，并且，它是开源的！！我们可以自行部署到我们的机器上(前提是你的显卡扛得住)。Stable Diffusion想对于Midjourney来说，其提示词更加复杂些，对使用者的要求更高。

官网地址: https://stablediffusionweb.com/

DALL-E 2

DALL-E 2是OpenAI公司开发的AI绘画工具，然而，DALL-E 2也是需要收费的。

官方地址: https://openai.com/product/dall-e-2

以上是主流的AI绘画类工具了，对于有专研精神的小伙伴们来说，Stable Difussion也许是个不错的选择。对于希望能快速画出好作品的同学来说，推荐使用Midjourney（要舍得钱）

在后续的课程中，我们会介绍主流的工具，帮助大家学会使用AI工具，为自己的工作提高效率，为生活添加乐趣。
